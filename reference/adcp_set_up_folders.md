# Set up folder structure for processing ADCP data

Set up folder structure for processing ADCP data

## Usage

``` r
adcp_set_up_folders(path = NULL, folder = NULL)
```

## Arguments

- path:

  Path to where the folder will be created. Default is the "Side Lobe
  Trimmed" folder on the Perennia shared drive.

- folder:

  Name of the top-level folder to be created. Default is
  todays_date_Process.

## Value

Returns folder structure that works well with CMAR ADCP processing
templates.
