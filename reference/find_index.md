# Index of first bin column

Index of first bin column

## Usage

``` r
find_index(dat_wide)
```

## Arguments

- dat_wide:

  Data frame of ADCP data, as exported from
  [`adcp_read_txt()`](https://dempsey-cmar.github.io/adcp/reference/adcp_read_txt.md).

## Value

Returns the index of the first column with bin data.
